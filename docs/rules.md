# Правила работы с проектом

- После любых изменений необходимо запускать тесты по инструкции из README.md, а также запускать проверку линтером.
- При изменении структуры `.env` (добавление/удаление/переименование переменных) необходимо синхронно обновлять `.env-example`.
- запускать тесты можно ТОЛЬКО ПО ИНСТРУКЦИИ из README.md
- При любых изменениях в проекте обязательно обновлять README.md.
