# Микросервис по форматированию сообщений для telegram
## Эндпоинты:
 - Эндпоинт для проверки жизни микросервиса, авторизация не трубуется
 - Эндпоинт, который получает текстовое сообщение в теле запроса в формате MarkDown и возвращает сообщение в формате,
  совместимостимом с Telegram.
  Возврат должен быть в виде массива, т.к. телеграм поддерживает ограниченную длину сообщений и длинное сообщение
  нужно корректно разбивать на несколько.
  Пример возврата:
  [
    {
      "text": "Форматированный текст",
    }
  ]
  Суть эндпоинта: нужно исключить не совместимые с Telegram символы, сделать корректную разбивку на сообщения при превышении
  допустимой длины.

## Технологии:
 - FastAPI
 - Pydantic
